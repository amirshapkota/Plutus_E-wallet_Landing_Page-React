## Technologies 
This landing page makes use of React, Bootstrap and few other technologies.

Note: This is just the landing page of the project.

## Setup 
> Make sure NPM is installed. 
1. Clone the repo
```bash
git clone 
```
2. Install all required React Plugins
```bash 
npm install
```
4. Run the Project
``` bash 
npm start
```
